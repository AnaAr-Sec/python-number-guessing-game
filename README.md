# 🎯 Number Guessing Game (Python)

This is a simple number guessing game developed in Python.

The program generates a random number between 1 and 100, and the user has 8 attempts to guess it.

---

## 🚀 Features

- User input handling
- Random number generation
- Input validation
- Error handling with try/except
- Conditional logic (if/elif/else)
- Loops (for)

---

## 🧠 Skills demonstrated

- Python fundamentals
- Control flow
- Exception handling
- Problem-solving logic

---

## 🎮 How it works

1. The program asks for your name
2. It generates a random number between 1 and 100
3. You have 8 attempts to guess the number
4. The program gives feedback:
   - Number too high
   - Number too low
   - Correct guess 🎉

---

## ▶️ How to run

1. Install Python
2. Run the script:

```bash
python guessing_game.py
```
